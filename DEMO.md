# Demo

Demo video and screenshots will be added upon project completion.

## Planned Demo Scenarios

1. **Agent Collection** — Show the agent collecting host metrics, tailing logs, and receiving OTel spans
2. **Pipeline E2E** — Inject metrics, query them via PromQL, and display on dashboard
3. **Failure Recovery** — Kill the ingestion backend, show zero data loss via disk buffer, then recover
4. **Alerting** — Inject a threshold breach, show alert firing and webhook notification
5. **Anomaly Detection** — Inject a synthetic spike into baseline data, show anomaly flagged on dashboard
6. **Trace Drill-Down** — Navigate from a metric spike to correlated traces and logs
